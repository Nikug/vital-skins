How to install

Vital allows using custom skins from any folder. I personally created a folder for skins in the Vital's user data folder (the same place where presets are stored).


Install using the following steps:

  1. Download a skin with the download button
  2. Move the file anywhere you want to
  3. Open Vital
  4. Go to the advanced tab, click the skin button that is in the bottom and select "Load custom skin..."
  5. Select the skin you want to use
  6. *happy Vital noises*
