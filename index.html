How to install

Vital allows using custom skins from any folder. I personally created a folder for skins in the user data folder.


Install using the following steps:

  1. Download the skin with the download button
  2. Move the file to your Vital skin folder
  3. Open Vital
  4. Go to the settings from some button
  5. Add the skin folder
  6. Go to the "some tab" in Vital
  7. From the bottom select the skin you want to use
